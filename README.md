# Mon App — Suivi & Communauté

Application Flutter de suivi d'évolution personnelle
avec espaces groupés et liens entre personnes.

## Stack
- Mobile : Flutter (iOS + Android)
- Backend : Node.js + Express
- Base de données : Supabase (PostgreSQL)
- Auth : Supabase Auth

## Structure
mon-app/
├── flutter_app/   → application mobile
├── backend/       → API REST
└── README.md
